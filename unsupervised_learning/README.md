# Unsupervised Learning

Implementations of unsupervised learning algorithms including clustering, dimensionality reduction, autoencoders, and generative models.

## Modules

| Directory | Description |
|-----------|-------------|
| [clustering](clustering/) | K-means, Gaussian Mixture Models (GMM), and agglomerative clustering |
| [dimensionality_reduction](dimensionality_reduction/) | Principal Component Analysis (PCA) from scratch using NumPy |

## Requirements

- Python 3.x
- NumPy
- scikit-learn
- Matplotlib
