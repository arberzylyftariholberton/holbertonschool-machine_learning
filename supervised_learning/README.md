# Supervised Learning

A collection of supervised learning projects implemented in Python using
NumPy, TensorFlow, and Keras. Each subdirectory is a self-contained module
covering a distinct topic in supervised machine learning.

## Modules

| Directory | Description |
|---|---|
| [`classification`](classification/) | Binary and multiclass neural networks built from scratch |
| [`cnn`](cnn/) | Convolutional layer forward/backward passes and LeNet-5 |
| [`deep_cnns`](deep_cnns/) | Deep CNN architectures: Inception, ResNet, DenseNet |
| [`error_analysis`](error_analysis/) | Confusion matrices, sensitivity, precision, F1 score |
| [`keras`](keras/) | High-level model building, training, and tuning with Keras |
| [`object_detection`](object_detection/) | YOLO-based object detection pipeline |
| [`optimization`](optimization/) | Gradient descent variants, Adam, batch norm, learning rate decay |
| [`regularization`](regularization/) | L2 regularization, dropout, early stopping, batch norm |
| [`RNNs`](RNNs/) | Recurrent networks: RNN, GRU, LSTM, bidirectional, encoder-decoder |
| [`time_series`](time_series/) | Forecasting with RNNs on Bitcoin price data |
| [`word_embeddings`](word_embeddings/) | Word2Vec, FastText, ELMo, and transformer embeddings |
| [`nlp_metrics`](nlp_metrics/) | BLEU score variants for NLP evaluation |
| [`qa_bot`](qa_bot/) | BERT-based extractive QA with semantic document retrieval |

## Requirements

- Python 3.9
- NumPy 1.25.2
- TensorFlow 2.15
- Keras (bundled with TensorFlow)

Additional per-module dependencies are listed in each subdirectory's README.
