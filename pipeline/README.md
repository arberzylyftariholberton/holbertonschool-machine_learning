# Pipeline

Data pipeline modules covering API consumption, data manipulation with Pandas, and image augmentation with TensorFlow.

## Modules

| Directory | Description |
|-----------|-------------|
| [apis](apis/) | REST API consumption using the SWAPI and SpaceX APIs |
| [pandas](pandas/) | DataFrame creation, transformation, and analysis using Pandas |
| [data_augmentation](data_augmentation/) | Image augmentation techniques using TensorFlow |

## Requirements

- Python 3.x
- requests
- pandas
- tensorflow
