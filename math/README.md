# Math

This directory contains foundational mathematics modules used throughout the machine learning curriculum.

## Modules

| Directory | Description |
|-----------|-------------|
| [calculus](calculus/) | Summation series and polynomial differentiation/integration |
| [linear_algebra](linear_algebra/) | Matrix operations, slicing, transposition, and multiplication using pure Python and NumPy |
| [plotting](plotting/) | Data visualization using Matplotlib — line plots, scatter plots, histograms, bar charts, and multi-plot figures |
| [probability](probability/) | Probability distributions: Poisson, Exponential, Normal, and Binomial |
| [convolutions_and_pooling](convolutions_and_pooling/) | Convolution and pooling operations implemented from scratch with NumPy |

## Requirements

- Python 3.x
- NumPy
- Matplotlib
