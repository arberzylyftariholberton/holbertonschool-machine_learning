# Linear Algebra

Implementations of linear algebra operations on matrices and vectors using both pure Python and NumPy.

## Tasks

| File | Function | Description |
|------|----------|-------------|
| `2-size_me_please.py` | `matrix_shape(matrix)` | Returns the shape of a nested Python list as a list of integers |
| `3-flip_me_over.py` | `matrix_transpose(matrix)` | Returns the transpose of a 2D matrix |
| `4-line_up.py` | `add_arrays(arr1, arr2)` | Element-wise addition of two arrays |
| `5-across_the_planes.py` | `add_matrices2D(mat1, mat2)` | Element-wise addition of two 2D matrices |
| `6-howdy_partner.py` | `cat_arrays(arr1, arr2)` | Concatenates two arrays |
| `7-gettin_cozy.py` | `cat_matrices2D(mat1, mat2, axis=0)` | Concatenates two 2D matrices along a given axis |
| `8-ridin_bareback.py` | `mat_mul(mat1, mat2)` | Performs matrix multiplication on two 2D matrices |
| `10-ill_use_my_scale.py` | `np_shape(matrix)` | Returns the shape of a NumPy array |
| `11-the_western_exchange.py` | `np_transpose(matrix)` | Transposes a NumPy array |
| `12-bracin_the_elements.py` | `np_elementwise(mat1, mat2)` | Element-wise add, subtract, multiply, divide on NumPy arrays |
| `13-cats_got_your_tongue.py` | `np_cat(mat1, mat2, axis=0)` | Concatenates two NumPy arrays along a given axis |
| `14-saddle_up.py` | `np_matmul(mat1, mat2)` | Matrix multiplication using `np.matmul` |
| `100-slice_like_a_ninja.py` | `np_slice(matrix, axes={})` | Slices a NumPy matrix along specific axes given as a dict |
| `101-the_whole_barn.py` | `add_matrices(mat1, mat2)` | Adds two matrices of any shape (pure Python) |
| `102-squashed_like_sardines.py` | `cat_matrices(mat1, mat2, axis=0)` | Concatenates matrices of any shape along a given axis |

## Requirements

- Python 3.x
- NumPy
