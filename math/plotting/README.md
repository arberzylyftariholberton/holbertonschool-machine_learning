# Plotting

Data visualization tasks using Matplotlib, progressing from basic plots to multi-subplot figures.

## Tasks

| File | Function | Description |
|------|----------|-------------|
| `0-line.py` | `line()` | Line plot of y = x³ for x in [0, 10] with a red solid line |
| `1-scatter.py` | `scatter()` | Scatter plot of men's height vs weight sampled from a multivariate normal distribution |
| `2-change_scale.py` | `change_scale()` | Line plot of exponential C-14 decay with a logarithmic y-axis |
| `3-two.py` | `two_plots()` | Dual-line plot comparing exponential decay of C-14 and Ra-226 |
| `4-frequency.py` | `frequency()` | Histogram of student grades with bins of width 10 |
| `5-all_in_one.py` | `all_in_one()` | Combined 3×2 subplot figure containing all five previous plots |
| `6-bars.py` | `bars()` | Stacked bar chart showing fruit quantities per person |
| `100-gradient.py` | `gradient()` | Scatter plot of mountain elevation using a color gradient and colorbar |
| `101-pca.py` | — | 3D PCA visualization of the iris dataset projected onto 3 principal components |

## Requirements

- Python 3.x
- NumPy
- Matplotlib
